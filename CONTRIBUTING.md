# Contributing

Thank you for your interest in contributing to the BitDevs Playbook by [Btrust](https://www.btrust.tech/). This Playbook is a living document and benefits from community feedback and shared experience.

## Who Can Contribute
- BitDevs organizers and volunteers
- Sponsors and partners
- Community members with relevant experience
- Contributors aligned with Bitcoin-only, open-source values

## What You Can Contribute
- Clarifications or improvements to existing sections
- Corrections (typos, broken links, formatting)
- New examples, templates, or best practices
- Feedback based on real-world organizing experience

## How to Contribute
You can contribute in one of two ways:

### Option 1: Email
Send suggestions or feedback to: **builders@btrust.tech**. Please include:
- Section or file name
- Clear description of the proposed change
- Reason for the suggestion

### Option 2: GitHub Pull Request
1. Fork the repository
2. Create a new branch from master for your changes
3. Make your edits with clear commit messages
4. Open a pull request describing:
   - What changed
   - Why it improves the Playbook

## Contribution Principles
All contributions must:
- Remain Bitcoin-only
- Be non-commercial and neutral
- Respect community autonomy
- Align with BitDevs principles and ethos

## Review Process
All submissions are reviewed by the Playbook maintainer and, when necessary, relevant organizers or sponsors before being merged.

Thank you for helping strengthen the BitDevs ecosystem.
